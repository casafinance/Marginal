Set Marginal as your default PDF app

Windows keeps this setting behind its own Settings page — here's the fastest way there:

  1. Click Open Settings below.
  2. In the search box at the top, type .pdf
  3. Click the app currently listed, then choose Marginal.

Or — a system dialog that sometimes lets you pick Marginal right away (not guaranteed on every setup).
